# Tech Stack

## Core
- **Runtime**: Bun (v1.3.3)
- **Monorepo**: Turborepo (v2.8.9)
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript

## Services & Libraries
- **Database**: MongoDB (Node.js Driver v7.1.0)
- **AI**: Google Gemini (`@google/generative-ai`)
- **Background Jobs**: Trigger.dev v3
- **Storage**: Vercel Blob
- **Styling**: Tailwind CSS 4, `clsx`, `tailwind-merge`
- **Icons**: Lucide React

## Development Tools
- **Linting**: ESLint
- **Formatting**: Prettier
- **Type Checking**: TypeScript
